# Beauty Manager Application


## Project Description

A mobile application designed for people who want to manage their visits to beauty and hair salons in a simple and fast way. The application allows you to browse services, make appointments and manage your favorite service providers in one place.
Many people encounter difficulties when making appointments - telephone contact, lack of information about available dates or hard-to-find reviews of services. The Beauty Manager application eliminates these problems by centralizing all the necessary functions in one, intuitive tool. The target group is mainly women who regularly use beauty services and value convenience and saving time.

## Project Installation

1.  Install the necessary dependencies using the command:
    `npm install`
2.  Start the project with the command:
   ` npm start` or `npx expo start`, if you have problems launching the program, use `npx expo start --tunnel`.
4.  Install the Expo Go app on your phone and scan the QR code using Expo Go (Android) or the camera app (iOS).
    Alternatively, follow the instructions provided in the console.
